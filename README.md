# NamLeGree - Antigravity Dashboard

Dự án hỗ trợ cá nhân hóa công việc tại Gree Vietnam.

## Cài đặt
1. Mở Terminal tại: `D:\NAMLE\GIT\Antigravity\namlegree`
2. Cài đặt thư viện: `pip install -r requirements.txt`
3. Chạy ứng dụng: `streamlit run app.py`

## Truy cập nhanh
- Local: `localhost:8501`
- Production: `leducnam.com/greevietnam`
